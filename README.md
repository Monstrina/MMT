# MMT

Это простой веб-инструмент (frontend-only), созданный для помощи в переводе и обновлении локализаций (`.json`) для модов Minecraft.

Главная особенность — **возможность "доперевода"**. В отличие от простой генерации файла, этот инструмент сравнивает `en_us.json` (оригинал) с `ru_ru.json` (существующий перевод), если он есть в `.jar` файле.

Затем он **генерирует промпт только для недостающих ключей**. Это избавляет от необходимости переводить весь мод заново при выходе обновления.

## Как использовать

1.  **Запустите:** Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).
2.  **Загрузите файл:** Перетащите `.jar` файл мода в специальную область или нажмите на нее, чтобы выбрать файл.
3.  **Скопируйте промпт**
4.  **Переведите:** Вставьте скопированный текст в ai чат бота(gemini, chatgpt и тд) и получите в ответ готовый переведенный JSON-блок.
5.  **Вставьте перевод:** Скопируйте переведенный JSON и вставьте его во второе текстовое поле (`Вставьте переведенный JSON`).
6.  **Сгенерируйте ZIP:** Нажмите кнопку "Сгенерировать ZIP перевода".
7.  **Готово:** Браузер скачает `.zip` архив с готовым переводом.

## Стек технологий

* **HTML5:** Структура страницы.
* **CSS3:** Стилизация (с использованием переменных CSS).
* **JavaScript (ES6+):** Вся логика на стороне клиента.
* **[JSZip](https://stuk.github.io/jszip/):** Библиотека для чтения `.jar` (которые являются ZIP-архивами) и создания новых `.zip` файлов прямо в браузере.
